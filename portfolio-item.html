WARLOCKS vs MONSTERS
Fantasy Battle Game

HTML, CSS & JavaScript

GitHub Repo Live version

This is the main layout of a fantasy battle game. It is built using JavaScript (ES6 modules), HTML5, and CSS3 to create an engaging user experience. Developed responsive, mobile-friendly UI utilizing flexbox and media queries, improving accessibility across different screen sizes. Achieved smooth state transitions and game-over animations by managing asynchronous delays. It was built as a part of Scrimba's Frontend Developer Career Path.
